## git ssh配置
- pkg install git 
- ssh-keygen 
- 私钥id-rsa 公钥id-rsa.pub 
## 链接远程仓库
- git remote remove origin
- git remote add origin git@github.com:fengshenme/java.git

<!-- 克隆仓库 -->
- git clone git@github.com:fengshenme/java.git
<!-- 提交到本地仓库 -->
- git commit -m "提交信息"
<!-- 推送远程仓库 -->
- git plush 
- git plush --set-upstream origin master
